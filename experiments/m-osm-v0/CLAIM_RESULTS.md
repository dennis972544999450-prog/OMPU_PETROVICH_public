# M-OSM Claim Results

These decisions apply to the frozen received draft identified in
`SOURCE_FINGERPRINT.md`. `Supported` means supported inside the tested fixture or
stated mathematical model, not a universal result about language models.

| ID | Decision | Evidence and boundary |
|---|---|---|
| MOSM-C01 | Conditional support | `Kc = 2(gamma + D)` holds for the stated infinite all-to-all noisy Kuramoto model with centered Lorentzian frequencies. It is not an agent-swarm threshold. |
| MOSM-C02 | Falsified | The draft equality gives `r = 1.6462`, outside `0 <= r <= 1`; simulation at the draft `K` gave about `0.9030`. The square-root law is local and proportional near onset. |
| MOSM-C03 | Narrow equivalence only | The modern Hopfield update matches attention under explicit key/value/projection identifications. General attention and cross-session memory do not follow. |
| MOSM-C04 | Falsified as latent recovery | A fresh isolated process recovered only explicit tube fields. Source-backed rehydration works because external evidence is read, not because a phase vector restores hidden state. |
| MOSM-C05 | Falsified as recovery | A digest verifies a supplied candidate; it does not recover an absent preimage or context without an external corpus or lookup. |
| MOSM-C06 | Falsified as written | Comparing dimensionless `r` to rate-valued `Kc` is invalid. Pure slowdown preserved `r` exactly; frequency narrowing can cross a threshold but is not period doubling. |
| MOSM-C07 | Label mechanism falsified | Post-hoc 5, 8.33, 12.5, and 25 Hz labels had no causal role and failed held-out prediction. A future real timing actuator remains untested. |
| MOSM-C08 | Falsified | The primary-source audit found no source-preserving software mapping for the Physarum, crow, or laser constants. Current prototype sensitivity to all 15 fields is zero because no consumer reads them. |
| MOSM-C09 | Falsified for the draft tube | Near equal input size, ordinary summary C1 retained the whole task semantically while tube-only C2 retained only 2/12 supported fields in all three blind runs. |
| MOSM-C10 | Supported as external retrieval | Reference-preserving and verified-source conditions preserved operational state. This is durable retrieval with precedence, not latent recall. |
| MOSM-C11 | Provisionally supported | The branch projection preserved 12/12 fields in one fixture. It needs independent fixtures and repeated runs before promotion. |
| MOSM-C12 | Falsified and replicated | The corrupted C4 packet claimed `r = 0.999`; two blind agents recovered 0/12 target fields. Self-reported coherence did not detect truth. |

## Promotion status

No oscillator-specific claim meets the five-part promotion rule in `CLAIMS.md`.
The only runnable result promoted to a prototype is an evidence-backed handoff
container with typed identity, correction precedence, bounded verified reads,
and explicit uncertainty. It is deliberately named by function, not by a claim
that text agents literally oscillate.
